# Excel Sales Analysis Practice Dataset

## Project Overview

This project contains an Excel practice dataset created to improve data analysis skills using Microsoft Excel. The workbook is designed for practising basic to intermediate Excel functions, lookup formulas, pivot tables, charts, and business analysis.

The dataset is based on mock sales/order data and is useful for building confidence in real-world Excel tasks commonly used in data analyst, business analyst, and reporting roles.

## File Included

```text
excel_practice_dataset.xlsx
```

## Workbook Structure

The Excel workbook contains multiple sheets:

### 1. Sales_Data

This sheet contains the main sales transaction data. It includes information such as:

* Order ID
* Order Date
* Customer Name
* Region
* Product ID
* Product Name
* Category
* Quantity
* Unit Price
* Sales
* Cost
* Profit
* Discount
* Sales Representative

This sheet is used for most analysis tasks.

### 2. Product_Lookup

This sheet contains product reference information. It can be used to practise lookup functions such as:

* XLOOKUP
* VLOOKUP
* INDEX MATCH

### 3. Region_Targets

This sheet contains regional sales targets. It can be used to compare actual sales performance against target sales.

### 4. Practice_Tasks

This sheet includes Excel practice questions ranging from basic to intermediate level.

### 5. Answer_Check

This sheet helps verify formulas and check whether the analysis results are correct.

## How to Use This Workbook

1. Open the Excel file.
2. Start with the `Sales_Data` sheet.
3. Complete the questions in the `Practice_Tasks` sheet.
4. Use formulas, pivot tables, and charts to answer the questions.
5. Check your results using the `Answer_Check` sheet.
6. Create a simple dashboard using charts and summary tables.
